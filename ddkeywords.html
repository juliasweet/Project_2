Disaster Declaration Table Data Fields

Below is a list outining the data fields within the Disaster Declaration table and a desciption and type for each item.


List of Data Fields for the Disaster Declaration Table
Name Title Type Description
disasternumber Disaster Number integer Number assigned to designate an event or incident declared as a disaster
declarationdate Declaration Date date The date the disaster was declaredThe date the disaster was declared
disastername Disaster Name varchar The proper name or description of the disaster, i.e. HURRICANE MARIA, FLOODING, SEVERE STORMS
incidentbegindate Incident Begin Date date The date the incident itself began
incidentenddate Incident End Date date The date the incident itseld ended
declarationtype Declaration Type varchar Defines whether a disaster is a Major Disaster Declaration, Emergency Declaration, Fire Management Assistance Declaration, or Fire Suppression Authorization.
statecode State Code varchar Two letter code for U.S. states, districts, and territories
statename State Name varchar The name of phrase describing the U.S. state, district, or territory
incidenttype Incident Type varchar The specific category for the disaster, i.e. Coastal Storm, Drought, Flood, Snow, Hurricane, Wildfire
